# Enchantments Enhanced

A Minecraft Forge mod that replaces the vanilla enchanting table screen with a deterministic
enchantment workbench. Built for Minecraft 26.1.2 and Forge 64.0.10.

Insert an item or book, select exact enchantment levels, review lapis and XP
costs, then apply the full selection in one server-validated transaction.

## Requirements

- Minecraft 26.1.2
- Minecraft Forge 64.0.10 or newer in the 64.x line
- Java 25
